3PL Central Assessment Test

In addition to sharing any projects or code that you have worked on throughout the last few years, we'd like for you to try the following programming challenge.

Build a web app to display meaningful data

Find a publicly available API that you can leverage to request some interesting data (e.g. Marvel , Hearthstone , OMDb Movie Info , League of Legends , etc.) then create a responsive web application to display the data in interesting ways. We'll let you interpret what "interesting" is exactly, but think of what may be useful for the user to learn by using your application and how that knowledge may change something they do. For example, you could build an app that allows a user to search for a movie and show its poster with relevant information for that movie.

A few requirements

  • Try to keep the amount of time you spend on the assessment under 4 hours. We don't want this to be a chore or a science project. The idea behind this is to give us a glimpse into what you're capable of, so have fun!
  • AngularJS

    We have provided a structure that mimics our work environment. This should give you an idea of how our code is structured and what you'd be working with on a day-to-day basis. Beyond that, it's up to you to decide how to implement and present your solution.

  • Have fun! Be creative, we're excited to see what you can do!
